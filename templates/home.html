This is a simple forum with "Inns" (a bit like subreddits) and "Solo" (for microblogging). It is a fork of freedit
Plebfreedit exists to demo one extra feature: access to the forum requires ownership of a single utxo greater than 500K sats.
You don't need to SPEND the utxo, or hold it, it just needs to exist. Moreover, which utxo is yours is kept hidden, using the power of zero knowledge proofs.
To generate the proof, use the tool here. To do this you'll need a "keyset file", a large file of all the taproot pubkeys in the current utxo set. You can either generate that yourself (which takes a long time, but see instructions here, or download the latest keyset file (they are generated every 1000 blocks) here.) Then paste the text created into the signup page here on this forum. If the proof is accepted, your username and password are permanently accepted.
